Welcome to A5C Documentation
==========================

A5C (agentic) is an AI-powered automation system that uses specialized agents to handle various aspects of software development and project management. It integrates with GitHub through GitHub Actions to provide continuous, automated assistance.

Quick Start
----------

To get started with A5C:

1. Use the `zero-to-demo <https://github.com/a5c-ai/zero-to-demo>`_ template
2. Write your project requirements in the GitHub project description
3. The project-seeder-agent will automatically analyze your requirements
4. Your project will be scaffolded with the appropriate structure and agents

Key Features
-----------

* **Agent-based Automation**: Specialized AI agents for different tasks
* **GitHub Integration**: Seamless integration with GitHub workflows
* **Flexible Triggering**: Event, mention, schedule, label, branch, and file-based triggers
* **Agent Collaboration**: Agents discover and work with each other
* **Extensible Architecture**: Add new capabilities through custom agents and MCP servers
